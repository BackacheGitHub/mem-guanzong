# 算术 · 奇数与偶数

## 基本定义

- **偶数 (Even Number)**：能被 2 整除的整数。形式为 $2k$（$k$ 为整数）。
    - 示例：..., -4, -2, 0, 2, 4, ...
- **奇数 (Odd Number)**：不能被 2 整除的整数。形式为 $2k + 1$（$k$ 为整数）。
    - 示例：..., -3, -1, 1, 3, 5, ...

## 判定方法

- **看末位**：末位是 0, 2, 4, 6, 8 的整数为偶数；末位是 1, 3, 5, 7, 9 的整数为奇数。
- **注意**：0 是偶数。

## 运算性质（核心考点）

| 运算 | 规则 | 示例 |
| :--- | :--- | :--- |
| **加减法** | 同性得偶，异性得奇 | $3+5=8$（偶），$3+2=5$（奇） |
| **乘法** | 偶 $\times$ 任何数 = 偶；奇 $\times$ 奇 = 奇 | $2 \times 3=6$（偶），$3 \times 5=15$（奇） |
| **幂运算** | 奇数的任意次幂仍为奇数；偶数的任意次幂仍为偶数 | $3^2=9$（奇），$2^3=8$（偶） |

## 关键结论

- **相邻整数**：必为一奇一偶，其和必为奇数，其积必为偶数。
- **奇数个奇数之和**：结果为奇数。
- **偶数个奇数之和**：结果为偶数。
- **若干个整数之积**：只要其中有一个偶数，积必为偶数。

## 典型例题

### 奇偶性分析

**例**：算式 $996 \times 567 - 5221 + 2332$ 的结果是奇数还是偶数？

**解**：利用奇偶性的运算性质进行分析，无需计算具体数值：
1. 996 是**偶数**，567 是**奇数**，5221 是**奇数**，2332 是**偶数**。
2. $996 \times 567$：偶数 × 奇数 = **偶数**。
3. (偶数) - 5221：偶数 - 奇数 = **奇数**。
4. (奇数) + 2332：奇数 + 偶数 = **奇数**。

**答案**：结果是 **奇数**。

### 代数与奇偶性

**例**：是否存在两个自然数，它们的和比它们的差多 5？

**解**：
1. 设两个自然数为 $x$ 和 $y$（$x \geq y$），则 $(x + y) - (x - y) = 5$。
2. 化简得 $2y = 5$，解得 $y = 2.5$。
3. 因为 $y$ 必须是自然数，2.5 不是整数。

**结论**：不存在这样的两个自然数。

> 备注：两个自然数的和与差具有相同的奇偶性，差必然是偶数，而 5 是奇数，逻辑上不成立。

### 奇偶性判定

**例**：如果 $m$、$n$ 是自然数，那么下列各数当中，一定是偶数的是（ ）。
A. $2m+3n$   B. $2m+3n^2$   C. $(m+n)^2$   D. $2m-4n$   E. $m-2n$

**解**：
- **A**：$2m$ 是偶数，$3n$ 奇偶性取决于 $n$，不一定是偶数。
- **B**：同理，$3n^2$ 取决于 $n$，不一定是偶数。
- **C**：若 $m+n$ 为奇数，则平方也为奇数。
- **D**：$2m$ 和 $4n$ 都是偶数，偶数 - 偶数 = **偶数**，一定是偶数。
- **E**：$2n$ 是偶数，结果奇偶性与 $m$ 相同。

**答案**：**D**

### 条件充分性判断

**例 (2012.13)**：已知 $m, n$ 是正整数，则 $m$ 是偶数。
(1) $3m + 2n$ 是偶数。
(2) $3m^2 + 2n^2$ 是偶数。

**解**：
1. **条件 (1)**：$2n$ 是偶数 → $3m$ 是偶数 → 因 3 是奇数，故 $m$ 必须是偶数。**充分**。
2. **条件 (2)**：$2n^2$ 是偶数 → $3m^2$ 是偶数 → $m^2$ 是偶数 → $m$ 必须是偶数。**充分**。

**答案**：**D**（两个条件都充分）

### 奇偶性应用：杯子问题

**例**：桌上放着七个杯子，三只杯口朝上，四只杯口朝下，每个人任意将杯子翻动四次。若干人翻动后，能否将七只杯子全变成杯口朝下？

**解**：利用**奇偶性不变性**原理：
1. 初始：朝上的杯子有 3 只（**奇数**）。
2. 目标：朝上的杯子有 0 只（**偶数**）。
3. 每次翻动 4 只杯子，变化量 $\Delta = 4 - 2k$（$k$ 为翻动的朝上杯子数），必然是偶数。
4. 因此朝上杯子数量的**奇偶性永远不变**，初始是奇数，不可能达到 0（偶数）。

**答案**：**不能**。

---

### 奇偶性应用：翻转问题

**例**：桌面上有 8 只杯子，将其中 3 只杯子翻转作为一次操作。8 只杯口朝上的杯子经 n 次操作后，杯口全部朝下，则 n 的最小值为（ ）。
A. 3   B. 4   C. 5   D. 6   E. 8

**解**：
1. 每只杯子必须被翻转**奇数次**，8 只杯子的总翻转次数须为偶数（8 个奇数之和）。
2. n 次操作总翻转次数为 **3n**，须为偶数 → **n 必须是偶数**。
3. 排除 A(3)、C(5)。n=4 时总次数 12 次，可分配为：6 只翻 1 次，2 只翻 3 次。

**答案**：**B**
