# 算术 · 比

## 定义

**比**表示两个量之间的倍数关系，用冒号或分数线表示。

- $a$ 与 $b$ 的比记作 $a : b$ 或 $\dfrac{a}{b}$
- $a$ 称为**前项**（比的前项），$b$ 称为**后项**（比的后项）
- $\dfrac{a}{b}$ 称为**比值**（比的结果）

**示例**：$6 : 4 = 3 : 2$，比值为 $\dfrac{3}{2} = 1.5$

**例如**：某班有男生 15 人，女生 10 人，男女的比是？

**解**：男女的比 = 15 : 10 = 3 : 2

## 比的基本性质

| 性质 | 说明 |
| ---- | ---- |
| 基本性质 | 比的前项和后项同时乘以或除以同一个非零数，比值不变 |
| 化简 | 将比化为最简整数比（前项与后项为互质整数） |
| 互为倒数 | $a : b = \dfrac{1}{b} : \dfrac{1}{a}$（前提：$a, b \neq 0$） |
| 等比性质 | 若 $\dfrac{a}{b} = \dfrac{c}{d} = k$，则 $\dfrac{a+c}{b+d} = \dfrac{a-c}{b-d} = k$ |

**示例**：$0.6 : 0.4 = 6 : 4 = 3 : 2$

**等比性质示例**：若 $a : b = c : d = 3 : 2$，则 $(a + c) : (b + d) =$ \_\_\_\_

**解**：由等比性质直接得 $(a + c) : (b + d) = 3 : 2$

**例如**：若 $\dfrac{x}{3} = \dfrac{y}{5}$，则 $\dfrac{x + y}{3 + 5} =$ \_\_\_\_

**解**：由等比性质：

$$
\dfrac{x + y}{3 + 5} = \dfrac{x}{3} = \dfrac{y}{5}
$$
> **技巧**：看到多个比相等（$\dfrac{a}{b} = \dfrac{c}{d}$），可直接用等比性质求和或差的比，无需分别求出各量。

## 化简比的方法

1. **整数比**：前项与后项同时除以它们的最大公约数（GCD）
   - $12 : 8 = 3 : 2$（同时除以 GCD(12, 8) = 4）

2. **分数比**：前项与后项同时乘以分母的最小公倍数（LCM）
   - $\dfrac{1}{2} : \dfrac{2}{3} = 3 : 4$（同时乘以 LCM(2, 3) = 6）

3. **小数比**：先化为整数比，再化简
   - $0.75 : 0.25 = 75 : 25 = 3 : 1$

4. **带单位的比**：单位统一后再化简
   - 3米 : 50厘米 = 300厘米 : 50厘米 = 6 : 1

**例如**：化简 $\dfrac{1}{6} : \dfrac{2}{9}$

**解**：同时乘以 LCM(6, 9) = 18

$$
\dfrac{1}{6} \times 18 : \dfrac{2}{9} \times 18 = 3 : 4
$$
> **易错点**：化简分数比时，不能只化简其中一个分数，必须前后项同时乘以相同的数。

**例如**：化简 $x : y = \dfrac{1}{5} : \dfrac{1}{6}$

**解**：同时乘以 LCM(5, 6) = 30

$$
\dfrac{1}{5} \times 30 : \dfrac{1}{6} \times 30 = 6 : 5
$$
> **错误做法**：只把 $\dfrac{1}{5}$ 化为整数，写成 $1 : \dfrac{1}{6}$，这是错误的！

**例如**：化简 0.75 : 2

**解**：化为整数比后化简

$$
0.75 : 2 = 75 : 200 = 3 : 8
$$
## 比与分数、除法的关系

| 概念 | 关系 | 说明 |
| ---- | ---- | ---- |
| 比 → 分数 | $a : b = \dfrac{a}{b}$（$b \neq 0$） | 比值就是分数 |
| 比 → 除法 | $a : b = a \div b$（$b \neq 0$） | 前项是被除数，后项是除数 |
| 三项关系 | 前项 = 后项 × 比值 | 用于求比中的未知项 |

**例如**：$( \quad ) \div 4 = \dfrac{9}{( \quad )} = 0.75 = ( \quad ) : 52 = ( \quad )\%$

**解**：从已知值 $0.75$ 出发，逐个推导

- $0.75 = \dfrac{3}{4}$，所以 $(3) \div 4 = 0.75$
- $\dfrac{9}{( \quad )} = \dfrac{3}{4}$，交叉相乘得 $3 \times ( \quad ) = 9 \times 4 = 36$，所以括号内为 $12$
- $( \quad ) : 52 = \dfrac{3}{4}$，则 $( \quad ) = 52 \times \dfrac{3}{4} = 39$
- $0.75 = 75\%$

$$
(3) \div 4 = \dfrac{9}{12} = 0.75 = (39) : 52 = (75)\%
$$
**例如**：如果 $5A = 2B$，那么 $A : B =$ \_\_\_\_

**解**：将等式变形为比的形式

$$
5A = 2B \implies \dfrac{A}{B} = \dfrac{2}{5}
$$
所以 $A : B = 2 : 5$

> **技巧**：看到 $mA = nB$，直接写 $A : B = n : m$（系数交叉对调）

**例如**：$\dfrac{1}{2} : \dfrac{1}{3} = ( \quad ) : \dfrac{1}{6}$

**解**：先化简左边的比

$$
\dfrac{1}{2} : \dfrac{1}{3} = \dfrac{1}{2} \times 6 : \dfrac{1}{3} \times 6 = 3 : 2
$$
再利用比的基本性质，后项从 2 变为 $\dfrac{1}{6}$，相当于除以 12：

$$
3 : 2 = \left(3 \div 12\right) : \left(2 \div 12\right) = \dfrac{1}{4} : \dfrac{1}{6}
$$
所以 $\dfrac{1}{2} : \dfrac{1}{3} = \left(\dfrac{1}{4}\right) : \dfrac{1}{6}$

## 连比

若 $a : b = m : n$ 且 $b : c = p : q$，则 $a : b : c$ 可以合并为连比。

**方法**：统一中间项（$b$）的份数。

**示例**：$a : b = 2 : 3$，$b : c = 4 : 5$
- 统一 $b$：取 LCM(3, 4) = 12
- $a : b = 8 : 12$，$b : c = 12 : 15$
- 连比：$a : b : c = 8 : 12 : 15$

**例如**：甲与乙的比是 $6 : 5$，乙与丙的比为 $4 : 3$，甲与丙的比是多少？

**解**：统一乙的份数，取 LCM(5, 4) = 20

- 甲 : 乙 = $6 : 5 = 24 : 20$
- 乙 : 丙 = $4 : 3 = 20 : 15$

连比：甲 : 乙 : 丙 = $24 : 20 : 15$

所以甲与丙的比为 $24 : 15 = 8 : 5$

**例如**：已知甲、乙两公司的利润之比为 $3 : 4$，甲丙两公司的利润之比为 $1 : 2$，若乙公司的利润为 3000 万元，则丙公司的利润为（　　）

A. 5000 万元　　B. 4500 万元　　C. 4000 万元　　D. 3500 万元　　E. 2500 万元

**解**：先统一甲的份数，取 LCM(3, 1) = 3

- 甲 : 乙 = $3 : 4$
- 甲 : 丙 = $1 : 2 = 3 : 6$

连比：甲 : 乙 : 丙 = $3 : 4 : 6$

乙占 4 份对应 3000 万元，每份为 $3000 \div 4 = 750$ 万元

丙占 6 份，利润为 $750 \times 6 = 4500$ 万元

**答案：B**

**例如**：某家庭在一年的总支出中，子女教育支出与生活资料支出的比为 $3 : 8$，文化娱乐支出与子女教育支出的比为 $1 : 2$，已知文化娱乐支出占家庭总支出的 $10.5\%$，则生活资料支出占家庭总支出的（　　）

A. 40%　　B. 42%　　C. 48%　　D. 56%　　E. 64%

**解**：先统一子女教育支出的份数，取 LCM(3, 2) = 6

- 子女教育 : 生活资料 = $3 : 8 = 6 : 16$
- 文化娱乐 : 子女教育 = $1 : 2 = 3 : 6$

连比：文化娱乐 : 子女教育 : 生活资料 = $3 : 6 : 16$

总份数 = $3 + 6 + 16 = 25$

文化娱乐占 3 份，对应总支出的 $10.5\%$，每份占总支出的 $10.5\% \div 3 = 3.5\%$

生活资料占 16 份，占总支出的 $16 \times 3.5\% = 56\%$

**答案：D**

**例如**：花园里有玫瑰、百合还有兰花，其中玫瑰和百合的朵数之比为 1:2，而百合和兰花的朵数之比为 4:3，如果玫瑰比兰花少 20 朵，那么玫瑰花有多少朵？

**解**：先统一百合的份数，取 LCM(2, 4) = 4

- 玫瑰 : 百合 = $1 : 2 = 2 : 4$
- 百合 : 兰花 = $4 : 3$

连比：玫瑰 : 百合 : 兰花 = $2 : 4 : 3$

设玫瑰、百合、兰花分别为 $2x, 4x, 3x$ 朵

由题意：玫瑰比兰花少 20 朵，即 $3x - 2x = 20$，解得 $x = 20$

玫瑰有 $2 \times 20 = 40$ 朵

**答案：40 朵**

> **技巧**：遇到两个比涉及三个量时，先统一中间项的份数，写出连比，再设未知数求解。

**解**：已知 $\dfrac{1}{a} : \dfrac{1}{b} : \dfrac{1}{c} = 3 : 4 : 5$，设公比为 $k$，则：

$$
\dfrac{1}{a} = 3k, \quad \dfrac{1}{b} = 4k, \quad \dfrac{1}{c} = 5k
$$
所以：

$$
a = \dfrac{1}{3k}, \quad b = \dfrac{1}{4k}, \quad c = \dfrac{1}{5k}
$$
求 $a : b : c$：

$$
a : b : c = \dfrac{1}{3k} : \dfrac{1}{4k} : \dfrac{1}{5k} = \dfrac{1}{3} : \dfrac{1}{4} : \dfrac{1}{5}
$$
同时乘以 LCM(3, 4, 5) = 60：

$$
a : b : c = 20 : 15 : 12
$$
> **技巧**：看到 $\dfrac{1}{a} : \dfrac{1}{b} : \dfrac{1}{c} = m : n : p$，则 $a : b : c = \dfrac{1}{m} : \dfrac{1}{n} : \dfrac{1}{p}$，再化为整数比即可

## 比例

**比例**表示两个比相等的式子。

若 $a : b = c : d$（即 $\dfrac{a}{b} = \dfrac{c}{d}$），则 $a, b, c, d$ 构成比例。

### 比例的基本性质（交叉相乘）

$$
a \times d = b \times c
$$
即**内项之积 = 外项之积**。

**示例**：$2 : 3 = 4 : 6$，验证：$2 \times 6 = 3 \times 4 = 12$

### 解比例

已知比例中的三个量，求第四个量。

**示例**：若 $x : 3 = 8 : 6$，则 $x = \dfrac{3 \times 8}{6} = 4$

## 正比例与反比例

| 类型 | 定义 | 公式 |
| ---- | ---- | ---- |
| 正比例 | 两个量的比值（商）为常数 | $\dfrac{y}{x} = k$（$k$ 为常数） |
| 反比例 | 两个量的乘积为常数 | $x \times y = k$（$k$ 为常数） |

**正比例示例**：速度一定时，路程与时间成正比
- $\dfrac{300\text{km}}{5\text{h}} = \dfrac{600\text{km}}{10\text{h}} = 60\text{km/h}$

**例如**：以匀速行驶，行驶 120km 用时 2 小时，行驶 200km 用时多久？

**解**：速度一定，路程与时间成正比

$$
\dfrac{120}{2} = \dfrac{200}{x}
$$
$$
x = \dfrac{200 \times 2}{120} = \dfrac{400}{120} = \dfrac{10}{3} \approx 3.33 \text{小时}
$$
**反比例示例**：路程一定时，速度与时间成反比
- $60 \times 5 = 120 \times 2.5 = 300$

**例如**：老王每天步行上下班，去的时候每秒走 1.8 米，回来的时候每秒走 1.2 米，上下班共用时 25 分钟，那么老王家与公司相距多少千米？

**解**：路程一定时，速度与时间成反比

速度比（去 : 回）= $1.8 : 1.2 = 3 : 2$

时间比（去 : 回）= $2 : 3$（反比）

总时间 = 25 分钟 = 1500 秒，总份数 = $2 + 3 = 5$

去程时间 = $1500 \times \dfrac{2}{5} = 600$ 秒

路程 = 速度 × 时间 = $1.8 \times 600 = 1080$ 米 = $1.08$ 千米

**答案：1.08 千米**

> **技巧**：路程一定时，速度与时间成反比。先求速度比，再用反比关系求时间比，最后结合总时间求解。

### 练习：判断以下两种量成什么比例关系？

1. 长方体的高一定，体积和底面积 → **正比例**（$V = S \times h$，高 $h$ 一定，$\dfrac{V}{S} = h$ 为常数）
2. 圆的面积和半径 → **不成比例**（$S = \pi r^2$，面积与半径的平方成正比，与半径本身不成正比）
3. 班级里总人数一定，班上男生人数和女生人数 → **不成比例**（男生 + 女生 = 总人数，是和一定，不是积或商一定）
4. 工作总量一定，工作效率和工作时间 → **反比例**（工作总量 = 效率 × 时间，积为常数）
5. 路程一定，速度和时间 → **反比例**（路程 = 速度 × 时间，积为常数）
6. 溶质一定，溶液和浓度 → **正比例**（浓度 = 溶质 ÷ 溶液，溶质一定时 $\dfrac{\text{浓度}}{\text{溶液}} = \text{溶质}$ 为常数）

### 练习题

一列货车和一列客车同时从甲、乙两地相对开出，货车与客车的速度比是 $4 : 5$，货车行驶 4 小时到达乙站，问题：

- 货车与客车的行驶时间比是 \_\_\_\_
- 客车行驶 \_\_\_\_ 小时到达甲站

**解**：路程一定时，速度与时间成反比

- 速度比 = $4 : 5$，所以时间比 = $5 : 4$
- 货车时间 = 4 小时，对应 5 份，每份 = $4 \div 5 = 0.8$ 小时
- 客车时间 = $0.8 \times 4 = 3.2$ 小时

## 比例分配（按比分配）

将总量按给定比例分配。

**方法**：总量 × 各部分占总量的比例

**示例**：将 120 按 $2 : 3 : 5$ 分配
- 总份数：$2 + 3 + 5 = 10$
- 第一部分：$120 \times \dfrac{2}{10} = 24$
- 第二部分：$120 \times \dfrac{3}{10} = 36$
- 第三部分：$120 \times \dfrac{5}{10} = 60$

**例如（1997 年真题）**：若某人以 1000 元购买 A、B、C 三种商品，且所有金额之比为 $1 : 1.5 : 2.5$，则他购买 A、B、C 三种商品的金额（单位：元）依次是（　　）

A. 100, 300, 600　　B. 150, 225, 400　　C. 150, 300, 550　　D. 200, 300, 500　　E. 200, 250, 550

**解**：总份数 = $1 + 1.5 + 2.5 = 5$

- A 的金额：$1000 \times \dfrac{1}{5} = 200$ 元
- B 的金额：$1000 \times \dfrac{1.5}{5} = 300$ 元
- C 的金额：$1000 \times \dfrac{2.5}{5} = 500$ 元

**答案：D**

**例如（2001 年真题）**：一公司向银行借款 34 万元，欲按 $\dfrac{1}{2} : \dfrac{1}{3} : \dfrac{1}{9}$ 的比例分配给下属甲、乙、丙三个车间进行技术改造，则甲车间应得（　　）

A. 17 万元　　B. 8 万元　　C. 12 万元　　D. 18 万元

**解**：先将分数比化为整数比，同时乘以 LCM(2, 3, 9) = 18：

$$
\dfrac{1}{2} : \dfrac{1}{3} : \dfrac{1}{19} = 9 : 6 : 2
$$
总份数 = $9 + 6 + 2 = 17$

甲车间应得：$34 \times \dfrac{9}{17} = 18$ 万元

**答案：D**

**例如**：有 429 名小学生参加冬令营，其中男生和女生的人数比为 7:6。后来又有一些女生报名参赛，这时男生和女生的人数比变成 11:10。请问：后来报名的女生有多少人？

**解**：先求原来男生和女生的人数

男生 : 女生 = 7 : 6，总份数 = 7 + 6 = 13

每份人数 = 429 ÷ 13 = 33

- 原来男生 = 7 × 33 = 231 人
- 原来女生 = 6 × 33 = 198 人

后来女生报名后，男生人数不变，男生 : 女生 = 11 : 10

新女生人数 = 231 ÷ 11 × 10 = 210 人

后来报名的女生 = 210 - 198 = 12 人

**答案：12 人**

> **技巧**：比例变化问题中，不变量是关键。本题男生人数不变，利用不变量和新比例求出变化后的女生人数，再减去原来的女生人数即可。

## 常见应用题型

### 工程问题

合作效率 = 各自效率之和

**示例**：甲单独做需 10 天，乙单独做需 15 天，合作需几天？
- 甲效率：$\dfrac{1}{10}$，乙效率：$\dfrac{1}{15}$
- 合作效率：$\dfrac{1}{10} + \dfrac{1}{15} = \dfrac{5}{30} = \dfrac{1}{6}$
- 合作时间：6 天

**练习题**：甲、乙、丙三人各自独立做同一件工作，分别用时 10 分钟、20 分钟、30 分钟，那么他们的效率比是多少？

**解**：工作总量一定时，效率与时间成反比

- 时间比 = $10 : 20 : 30 = 1 : 2 : 3$
- 效率比 = $1 : \dfrac{1}{2} : \dfrac{1}{3}$
- 同时乘以 LCM(1, 2, 3) = 6，得效率比 = $6 : 3 : 2$

### 浓度问题

浓度 = $\dfrac{\text{溶质质量}}{\text{溶液质量}} \times 100\%$

**示例**：200g 15% 的盐水中加入 50g 水，浓度变为？
- 溶质质量：$200 \times 15\% = 30$g
- 新浓度：$\dfrac{30}{200 + 50} \times 100\% = 12\%$

**例如**：要将浓度为 25% 的酒精溶液 1020 克，配制成浓度为 17% 的酒精溶液，需加水多少克？

**解**：稀释问题中，溶质（酒精）的质量不变

溶质质量 = $1020 \times 25\% = 255$ 克

设需加水 $x$ 克，则：

$$
\dfrac{255}{1020 + x} = 17\%
$$
$$
1020 + x = \dfrac{255}{0.17} = 1500
$$
$$
x = 1500 - 1020 = 480 \text{ 克}
$$
**答案：480 克**

> **技巧**：稀释/加水问题，抓住"溶质不变"这一关键。加水前后酒精质量相等，据此列方程求解。

### 利润问题

利润率 = $\dfrac{\text{利润}}{\text{成本}} \times 100\%$，利润 = 售价 − 成本

**示例**：商品成本 80 元，售价 100 元，利润率？
- 利润率：$\dfrac{100 - 80}{80} \times 100\% = 25\%$

**例如**：某商品的成本利润率为 12%，若其成本降低 20% 而售价不变，则利润率为（　　）

A. 32%　　B. 35%　　C. 40%　　D. 45%　　E. 48%

**解**：设原成本为 100

原售价 = $100 \times (1 + 12\%) = 112$

新成本 = $100 \times (1 - 20\%) = 80$

售价不变，新利润 = $112 - 80 = 32$

新利润率 = $\dfrac{32}{80} \times 100\% = 40\%$

**答案：C**

> **技巧**：利润率变化问题，关键是抓住"售价不变"或"成本不变"等不变量，分别算出变化前后的成本和售价，再求新利润率。

### 追及问题

追及距离 = 速度差 × 追及时间

**例如**：甲乙两人从同一地方出发，甲先出发 10 分钟，若乙跑步追赶甲，则 10 分钟可追上，若乙骑车追赶甲，每分钟比跑步多行 100 米，则 5 分钟可追上，那么甲每分钟走的距离为（　　）

A. 50m　　B. 75m　　C. 100m　　D. 125m　　E. 150m

**解**：设甲速度为 $v$ 米/分，乙跑步速度为 $u$ 米/分

追及距离 = 甲先走 10 分钟的路程 = $10v$

- 跑步追及：$10v = (u - v) \times 10$，即 $v = u - v$，得 $u = 2v$
- 骑车追及：乙骑车速度 = $u + 100$，$10v = (u + 100 - v) \times 5$

将 $u = 2v$ 代入第二个方程：

$$
10v = (2v + 100 - v) \times 5 = (v + 100) \times 5
$$
$$
10v = 5v + 500 \implies 5v = 500 \implies v = 100
$$
**答案：C**
