# 数学基础考纲

## 考试概览

| 项目 | 内容 |
|------|------|
| 满分 | 75分 |
| 题量 | 25道单选题 |
| 考试时间 | 与管综其他部分合计3小时 |

---

## 题型分布

| 题型 | 题量 | 分值 |
|------|------|------|
| 问题求解 | 15题 | 45分（每题3分） |
| 条件充分性判断 | 10题 | 30分（每题3分） |

---

## 考查范围（四大模块）

### 算术（约7题）

| 考点 | 内容 | 详情 |
|------|------|------|
| 整除 | 整除、公倍数、公约数、奇数、偶数、质数、合数 | [整除](算术-整除.md) · [奇数与偶数](算术-奇数与偶数.md) · [质数和合数](算术-质数和合数.md) |
| 实数 | 运算技巧、绝对值 | |
| 分数与比例 | 分数、百分数、比例应用题 | [比](算术-比.md) |
| 应用题 | 行程问题、工程问题、浓度问题、利润问题等 | |

### 代数（约6-7题）

| 考点 | 内容 |
|------|------|
| 整式与分式 | 乘法公式、因式分解、通分约分 |
| 一元二次 | 二次函数（图像）、二次方程、二次不等式（三个"二次"关系） |
| 不等式 | 绝对值不等式、均值不等式 |
| 数列 | 等差数列、等比数列性质和公式 |

### 几何（约6-7题）

| 考点 | 内容 |
|------|------|
| 平面几何 | 三角形、四边形、圆和扇形的性质与面积 |
| 空间几何体 | 长方体、正方体、圆柱体、球体的表面积和体积 |
| 平面解析几何 | 斜率公式、中点坐标、两点间距离、点到直线距离 |

### 数据分析（约5-6题）

| 考点 | 内容 |
|------|------|
| 排列组合 | 计数原理、排列数、组合数 |
| 数据分析 | 平均值、方差、标准差 |
| 概率 | 古典概率、事件概率公式 |

---

## 考试特点

- **内容范围**：小学、初中、高中的初等数学，不涉及高等数学
- **难度定位**：低于高考难度
- **考查重点**：运算能力、逻辑推理能力、解题速度
- **难点提醒**：题量大、时间紧，需掌握快速解题技巧（特值法、排除法等）
