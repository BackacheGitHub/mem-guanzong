# 算术 · 整除

## 定义

若整数 \(a\) 除以非零整数 \(b\)，商为整数且余数为零，则称 \(a\) 能被 \(b\) 整除，记作 \(b \mid a\)。

- \(a\) 称为 \(b\) 的**倍数**
- \(b\) 称为 \(a\) 的**约数**（因数）

**示例**：\(12 \div 3 = 4\)，余数为0，所以 \(3 \mid 12\)

## 整除的性质

| 性质 | 说明 |
| ---- | ---- |
| 传递性 | 若 \(a \mid b\) 且 \(b \mid c\)，则 \(a \mid c\) |
| 线性组合 | 若 \(a \mid b\) 且 \(a \mid c\)，则 \(a \mid (mb + nc)\)（\(m,n\) 为整数） |
| 加减性 | 若 \(a \mid b\) 且 \(a \mid c\)，则 \(a \mid (b + c)\) 且 \(a \mid (b - c)\) |

## 常见整除判定规则

| 除数 | 判定规则 |
|------|----------|
| 2 | 末位是偶数（0, 2, 4, 6, 8） |
| 3 | [各位数字之和能被3整除](能被3整除的判定规则证明.md) |
| 4 | 末两位能被4整除 |
| 5 | 末位是0或5 |
| 6 | 同时能被2和3整除 |
| 8 | 末三位能被8整除 |
| 9 | 各位数字之和能被9整除 |
| 10 | 末位是0 |
| 11 | 奇数位数字之和与偶数位数字之和的差能被11整除 |
| 25 | 末两位能被25整除（即00, 25, 50, 75） |
| 125 | 末三位能被125整除（即000, 125, 250, 375, 500, 625, 750, 875） |

## 整除的传递与包含关系

- **能被 4 整除 $\implies$ 能被 2 整除**：因为 $4 = 2 \times 2$。
- **能被 8 整除 $\implies$ 能被 4 整除 $\implies$ 能被 2 整除**：因为 $8 = 2^3$。
- **能被 125 整除 $\implies$ 能被 25 整除 $\implies$ 能被 5 整除**：因为 $125 = 5^3$。

## 公倍数与公约数

> **术语缩写说明**：
> - **GCD**：Greatest Common Divisor（最大公约数）
> - **LCM**：Least Common Multiple（最小公倍数）

### 基本定义

- **公约数**：两个或多个整数公有的约数。
- **最大公约数（GCD）**：公约数中最大的一个。
- **公倍数**：两个或多个整数公有的倍数。
- **最小公倍数（LCM）**：公倍数中最小的一个正整数。

### 求法与公式

- **质因数分解法**：
    - 分解质因数后，GCD 取各质因数的**最小指数**之积。
    - LCM 取各质因数的**最大指数**之积。
    - *示例*：\(A = 2^2 \cdot 3^1\), \(B = 2^1 \cdot 3^3\)
        - \(GCD(A, B) = 2^1 \cdot 3^1 = 6\)
        - \(LCM(A, B) = 2^2 \cdot 3^3 = 108\)
- **短除法（最常用、最快）**：
    - 同时除以公有的质因数，直到商互质为止。
    - **GCD** = 所有公有质因数之积。
    - **LCM** = 所有公有质因数与最后互质商的乘积。
    - *示例*：
        ```
          2 | 24, 18
          ------------
          3 | 12, 9
          ------------
             4, 3
        ```
        - GCD = \(2 \times 3 = 6\)
        - LCM = \(2 \times 3 \times 4 \times 3 = 72\)
- **辗转相除法（欧几里得算法）**：
    - 利用 \(GCD(a, b) = GCD(b, a \pmod b)\) 递归求解，效率极高。
- **核心关系式**：
    - \(a \times b = GCD(a, b) \times LCM(a, b)\)

### 关键性质（MEM 考点）

- **互质（Coprime）**：
    - **定义**：若两个整数 \(a, b\) 的最大公约数为 1，即 \(GCD(a, b) = 1\)，则称这两个数互质（也称"互素"）。
    - **判定**：两数除了 1 之外没有其他公因数。
    - **常见结论**：
        1. 任意两个不同的**质数**一定互质。
        2. 若 \(a, b\) 互质，则 \(LCM(a, b) = a \times b\)。
        3. 若 \(a \mid c\) 且 \(b \mid c\)，且 \(a, b\) 互质，则 \(a \times b \mid c\)。
    - **示例**：8 和 15 互质（\(GCD(8, 15)=1\)），虽然 8 和 15 都是合数。
- **整除关系**：
    - 若 \(a \mid b\)，则 \(GCD(a, b) = a\) 且 \(LCM(a, b) = b\)。
    - \(GCD(a, b) \mid a\) 且 \(GCD(a, b) \mid b\)。
    - \(a \mid LCM(a, b)\) 且 \(b \mid LCM(a, b)\)。
- **倍数关系**：
    - 若 \(GCD(a, b) = d\)，则 \(a = md, b = nd\)，且 \(GCD(m, n) = 1\)。
    - \(LCM(a, b)\) 是 \(a\) 和 \(b\) 的公倍数中最小的，且 \(LCM(a, b) = k \cdot a = t \cdot b\)。

## 典型例题

### GCD 与 LCM 基础计算

1. **6 和 9**：
    - \(GCD(6, 9) = 3\)（公约数有1, 3）
    - \(LCM(6, 9) = 18\)（公倍数有18, 36...）
2. **3 和 5**：
    - \(GCD(3, 5) = 1\)（互质）
    - \(LCM(3, 5) = 15\)
3. **2 和 6**：
    - \(GCD(2, 6) = 2\)（整除关系：\(2 \mid 6\)）
    - \(LCM(2, 6) = 6\)
4. **16 和 24**：
    - \(16 = 2^4, 24 = 2^3 \times 3\)
    - \(GCD(16, 24) = 2^3 = 8\)
    - \(LCM(16, 24) = 2^4 \times 3 = 48\)

### GCD/LCM 在分数中的应用

**约分（利用 GCD）**：
- \(\frac{28}{36} = \frac{28 \div 4}{36 \div 4} = \frac{7}{9}\)
- \(\frac{35}{24} = \frac{35}{24}\)（互质，已是最简）
- \(\frac{42}{24} = \frac{42 \div 6}{24 \div 6} = \frac{7}{4}\)
- \(\frac{38}{57} = \frac{38 \div 19}{57 \div 19} = \frac{2}{3}\)
- \(\frac{91}{84} = \frac{91 \div 7}{84 \div 7} = \frac{13}{12}\)

**通分（利用 LCM）**：
- \(\frac{3}{4}, \frac{2}{5} \rightarrow LCM(4, 5)=20 \rightarrow \frac{15}{20}, \frac{8}{20}\)
- \(\frac{1}{4}, \frac{2}{6}, \frac{5}{8} \rightarrow LCM(4, 6, 8)=24 \rightarrow \frac{6}{24}, \frac{8}{24}, \frac{15}{24}\)
- \(\frac{1}{2}, \frac{3}{4}, \frac{2}{5}, \frac{7}{10} \rightarrow LCM(2, 4, 5, 10)=20 \rightarrow \frac{10}{20}, \frac{15}{20}, \frac{8}{20}, \frac{14}{20}\)

### GCD/LCM 综合练习

1. **18 和 24**：
    - \(GCD(18, 24) = 6\)
    - \(LCM(18, 24) = 72\)
2. **27 和 36**：
    - \(GCD(27, 36) = 9\)（两者都是9的倍数）
    - \(LCM(27, 36) = 108\)
3. **28 和 72**：
    - \(28 = 2^2 \times 7, 72 = 2^3 \times 3^2\)
    - \(GCD(28, 72) = 2^2 = 4\)
    - \(LCM(28, 72) = 2^3 \times 3^2 \times 7 = 504\)
4. **12, 30, 48 的最大公因数 (GCD)**：
    - \(12 = 2^2 \times 3\)
    - \(30 = 2 \times 3 \times 5\)
    - \(48 = 2^4 \times 3\)
    - \(GCD = 2^1 \times 3^1 = 6\)

### GCD/LCM 应用题

**例（公倍数应用）**：有一个电子钟，每走9分钟亮一次灯，每到整点响一次铃。中午12点整，电子钟既响铃又亮灯。问：下一次既响铃又亮灯是几点钟？

**解**：
1. **分析周期**：亮灯周期 9 分钟，响铃周期 60 分钟。
2. **求再次重合时间**：问题转化为求 9 和 60 的 **LCM**。
    ```
      3 | 9, 60
      ---------
         3, 20
    ```
    \(LCM(9, 60) = 3 \times 3 \times 20 = 180\)（分钟）= 3 小时。
3. **计算时刻**：从 12:00 开始，经过 3 小时，即为 **15:00**（下午3点）。

**答案**：下一次既响铃又亮灯是 **15:00**。

**例（GCD 应用）**：将一块长 90 厘米，宽 42 厘米的长方形铁板，剪成面积相等的小正方形而无剩余，问至少可以剪出多少块小正方形？

**解**：
1. **确定边长**：边长必须是 90 和 42 的**公约数**，要使块数最少则取 **GCD**。
2. **计算 GCD**：
    - \(90 = 2 \times 3^2 \times 5\)
    - \(42 = 2 \times 3 \times 7\)
    - \(GCD = 2 \times 3 = 6\)（厘米）
3. **计算块数**：\(90 \div 6 = 15\)，\(42 \div 6 = 7\)，总计 \(15 \times 7 = 105\)（块）。

**答案**：至少可以剪出 **105** 块小正方形。

### 整除判定应用

**例**：从 1 到 100 的整数中任取 1 个数，则该数能被 5 或 7 整除的概率为多少？

**解**：
1. 总样本数 \(n = 100\)。
2. 容斥原理：
    - \(|A|\)（被5整除）= \(\lfloor 100 \div 5 \rfloor = 20\)
    - \(|B|\)（被7整除）= \(\lfloor 100 \div 7 \rfloor = 14\)
    - \(|A \cap B|\)（被 LCM(5,7)=35 整除）= \(\lfloor 100 \div 35 \rfloor = 2\)
    - 符合条件 = \(20 + 14 - 2 = 32\)
3. \(P = 32 \div 100 = 0.32\)

**答案**：**D. 0.32**

**例**：张三给45名工人发放等额工资，总金额显示为 $67x8y$ 元（$x$、$y$ 为被遮挡数字），每名工人工资相同且为整数元，问每名工人的工资可能是多少？

**解**：
1. 总金额 $N$ 能被 45 整除。因 $45 = 5 \times 9$ 且 $GCD(5,9)=1$，$N$ 须**同时被 5 和 9 整除**。
2. **被 5 整除**：末位 $y$ 为 0 或 5。
3. **被 9 整除**：各位数字之和 $S = 6 + 7 + x + 8 + y = 21 + x + y$ 须是 9 的倍数。
4. 分类讨论：
    - $y = 0$：$S = 21 + x$，取 $x = 6$（和为27），$N = 67680$，工资 = $67680 \div 45 = 1504$（元）。
    - $y = 5$：$S = 26 + x$，取 $x = 1$（和为27），$N = 67185$，工资 = $67185 \div 45 = 1493$（元）。

**答案**：每名工人的工资可能是 **1504元** 或 **1493元**。

### 整除综合练习

**例1**：一个三位数，各位数字之和为12，能被3整除，求这样的三位数共有多少个？

**解**：各位数字之和为12，12能被3整除，因此满足"各位数字之和为12"的三位数必然能被3整除。问题转化为：各位数字之和为12的三位数有多少个？

设三位数为 \(\overline{abc}\)，则 \(a + b + c = 12\)，其中 \(1 \leq a \leq 9\)，\(0 \leq b, c \leq 9\)。

令 \(a' = a - 1\)，则 \(a' + b + c = 11\)，其中 \(0 \leq a' \leq 8\)，\(0 \leq b, c \leq 9\)。

- 无限制时非负整数解：\(C_{13}^{2} = 78\)
- 排除 \(a' \geq 9\)：\(C_{4}^{2} = 6\)
- 排除 \(b \geq 10\)：\(C_{3}^{2} = 3\)
- 排除 \(c \geq 10\)：\(C_{3}^{2} = 3\)

**答案**：\(78 - 6 - 3 - 3 = 66\) 个

**例2**：已知三位数 \(23x\)，当 \(x\) 为何值时，该数能分别被 3, 4, 5 整除？

**解**：
1. **能被3整除**：\(2+3+x=5+x\) 是3的倍数 → \(x\) 可填：**1, 4, 7**。
2. **能被4整除**：末两位 \(3x\) 是4的倍数 → \(x\) 可填：**2, 6**。
3. **能被5整除**：末位是0或5 → \(x\) 可填：**0, 5**。

> **注**：若要求**同时**被这三个数整除，则须被 \(LCM(3, 4, 5) = 60\) 整除。230\~239 之间没有 60 的倍数，无共同解。

---

**例3**：已知三位数 \(173x\)，当 \(x\) 为何值时，该数能分别被 8, 9 整除？

**解**：
1. **能被8整除**：末三位 \(173x\) 是8的倍数 → \(1736 \div 8 = 217\) → 填 **6**。
2. **能被9整除**：\(1+7+3+x=11+x\) 是9的倍数 → \(x=7\) 时 \(11+7=18\) → 填 **7**。

> **注**：若要求同时被 8 和 9 整除（即被 \(LCM(8, 9) = 72\) 整除），1730\~1739 之间没有 72 的倍数，无共同解。

**例4**：给出一组数：3124, 312, 3823, 45235, 5289, 5588, 661, 7314。请指出哪些能被 4 整除，哪些能被 3 整除？

**解**：
1. **能被 4 整除**（末两位能被4整除）：**3124**（24÷4=6）、**312**（12÷4=3）、**5588**（88÷4=22）。
2. **能被 3 整除**（数字和能被3整除）：**312**（3+1+2=6）、**5289**（5+2+8+9=24）、**7314**（7+3+1+4=15）。

### 判断练习

判断下列数能否被 4, 8, 25, 125, 3, 9 整除：
`23487, 3568, 8875, 6765, 5880, 7538, 198954, 6512, 93625, 864, 407`

**判定结果：**

- **能被 25 整除**（末两位）：**8875, 93625**
- **能被 125 整除**（末三位）：**8875, 93625**
- **能被 3 整除**（数字和）：**23487, 6765, 5880, 198954, 864**
- **能被 9 整除**（数字和）：**198954, 864**
