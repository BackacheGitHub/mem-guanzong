# AGENTS.md

## What this repo is

Chinese-language markdown documents about mathematics fundamentals and MEM (Master of Engineering Management) exam preparation. No code, no build system, no CI.

## Project structure

```
mem-guanzong/
├── 数学基础考纲.md           # Exam outline (overview + links to knowledge point files)
├── AGENTS.md
├── 算术/                    # Arithmetic module
│   ├── 整除.md              # Divisibility rules, GCD/LCM
│   ├── 奇数与偶数.md        # Odd/even number properties
│   ├── 质数和合数.md        # Prime/composite numbers
│   ├── 比.md                # Ratio and proportion
│   └── 能被3整除的判定规则证明.md  # Proof of divisibility rule for 3
├── 代数/                    # Algebra module
│   └── 绝对值基础.md        # Absolute value basics
├── 几何/                    # Geometry module (empty)
├── 数据分析/                # Data analysis module (empty)
├── 可视化/                  # Visualizations
│   ├── 绝对值方程可视化.html # Absolute value equation visualization
│   └── 绝对值函数图像.html  # Absolute value function graph
└── 报考指南/                # Registration guide
    └── MEM报考指南.md        # MEM exam registration guide
```

## Conventions

- All content is written in Chinese (Simplified).
- Files are standalone markdown — no cross-references, no shared includes.
- When editing, preserve the existing tone and academic style.
