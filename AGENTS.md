# AGENTS.md

## What this repo is

Chinese-language markdown documents about mathematics fundamentals and MEM (Master of Engineering Management) exam preparation. No code, no build system, no CI.

## Content files

- `数学基础考纲.md` — Exam outline (overview + links to knowledge point files)
- `算术-整除.md` — Divisibility rules, GCD/LCM, worked examples
- `算术-奇数与偶数.md` — Odd/even number properties and examples
- `算术-质数和合数.md` — Prime/composite number definitions and properties
- `能被3整除的判定规则证明.md` — Proof of divisibility rule for 3 (cross-referenced from 整除)
- `MEM报考指南.md` — MEM exam registration guide

## Conventions

- All content is written in Chinese (Simplified).
- Files are standalone markdown — no cross-references, no shared includes.
- When editing, preserve the existing tone and academic style.
