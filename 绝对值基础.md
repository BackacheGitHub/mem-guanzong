# 绝对值基础

## 定义

**绝对值**表示一个数在数轴上到原点的距离，是一个非负数。

$$
|a| = \begin{cases}
a, & a > 0 \\
0, & a = 0 \\
-a, & a < 0
\end{cases}
$$
**几何意义**：$|a|$ 表示数轴上点 $a$ 到原点的距离。

**示例**：
- $|5| = 5$
- $|-3| = 3$
- $|0| = 0$

## 绝对值的性质

| 性质 | 说明 |
| ---- | ---- |
| 非负性 | $|a| \geq 0$，绝对值永远是非负数 |
| 自反性 | $|a| = |-a|$，互为相反数的两个数绝对值相等 |
| 乘法 | $|a \times b| = |a| \times |b|$ |
| 除法 | $\left|\dfrac{a}{b}\right| = \dfrac{|a|}{|b|}$（$b \neq 0$） |

**例如**：$|-5| = |5| =$ \_\_\_\_

**解**：由自反性，$|-5| = |5| = 5$

## 绝对值的化简

### 去绝对值符号

根据绝对值内表达式的正负，去掉绝对值符号：

$$
|x| = \begin{cases}
x, & x \geq 0 \\
-x, & x < 0
\end{cases}
$$
**例如**：化简 $|x - 3|$

**解**：
- 当 $x \geq 3$ 时，$x - 3 \geq 0$，所以 $|x - 3| = x - 3$
- 当 $x < 3$ 时，$x - 3 < 0$，所以 $|x - 3| = -(x - 3) = 3 - x$

### 分段讨论

遇到绝对值方程或不等式时，需要根据绝对值内表达式的正负分段讨论。

**例如**：解方程 $|x - 2| = 3$

**解**：分两种情况

情况一：$x - 2 = 3$，解得 $x = 5$

情况二：$x - 2 = -3$，解得 $x = -1$

所以 $x = 5$ 或 $x = -1$

## 绝对值不等式

### 基本不等式

$$
|x| < a \iff -a < x < a \quad (a > 0)
$$
$$
|x| > a \iff x < -a \text{ 或 } x > a \quad (a > 0)
$$
**例如**：解不等式 $|x - 1| < 4$

**解**：由基本不等式

$$
-4 < x - 1 < 4
$$
$$
-3 < x < 5
$$
**例如**：解不等式 $|2x + 3| \geq 5$

**解**：由基本不等式

$$
2x + 3 \leq -5 \text{ 或 } 2x + 3 \geq 5
$$
$$
2x \leq -8 \text{ 或 } 2x \geq 2
$$
$$
x \leq -4 \text{ 或 } x \geq 1
$$
## 绝对值的几何应用

### 数轴上的距离

数轴上两点 $a$ 和 $b$ 之间的距离为 $|a - b|$。

**例如**：数轴上点 $A$ 表示 $-2$，点 $B$ 表示 $3$，则 $AB =$ \_\_\_\_

**解**：$AB = |3 - (-2)| = |5| = 5$

**例如**：已知 $|x - 1| + |x + 2| = 5$，求 $x$ 的值。

**解**：$|x - 1| + |x + 2|$ 表示数轴上点 $x$ 到点 $1$ 和点 $-2$ 的距离之和。

点 $1$ 和点 $-2$ 之间的距离为 $|1 - (-2)| = 3$

要使距离之和为 $5 > 3$，点 $x$ 必须在区间 $[-2, 1]$ 之外。

- 当 $x > 1$ 时：$(x - 1) + (x + 2) = 5$，解得 $x = 2$
- 当 $x < -2$ 时：$-(x - 1) - (x + 2) = 5$，解得 $x = -3$

所以 $x = 2$ 或 $x = -3$

## 常见题型

### 求最值

**例如**：求 $|x - 1| + |x + 3|$ 的最小值。

**解**：$|x - 1| + |x + 3|$ 表示点 $x$ 到点 $1$ 和点 $-3$ 的距离之和。

由几何意义，当点 $x$ 在 $[-3, 1]$ 之间时，距离之和最小，最小值为：

$$
|1 - (-3)| = 4
$$
**答案：最小值为 4**

### 绝对值三角不等式

$$
\big||a| - |b|\big| \leq |a \pm b| \leq |a| + |b|
$$
**例如**：已知 $|a| = 3$，$|b| = 5$，求 $|a + b|$ 的取值范围。

**解**：由三角不等式

$$
\big||a| - |b|\big| \leq |a + b| \leq |a| + |b|
$$
$$
|3 - 5| \leq |a + b| \leq 3 + 5
$$
$$
2 \leq |a + b| \leq 8
$$
## 易错点

1. **$|a| = a$ 不一定成立**：当 $a < 0$ 时，$|a| = -a$
2. **$|a| = -a$ 不一定成立**：当 $a > 0$ 时，$|a| = a$
3. **$\sqrt{a^2} = |a|$**：不是 $a$
4. **绝对值方程要分段讨论**：不能直接去掉绝对值符号

**例如**：判断 $\sqrt{(-3)^2} = -3$ 是否正确。

**解**：$\sqrt{(-3)^2} = \sqrt{9} = 3$，不等于 $-3$，所以错误。

正确结论：$\sqrt{a^2} = |a|$，不是 $a$
